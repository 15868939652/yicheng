【AI高频词黑名单v1.0】

一、开场套话（立即删除）
- 随着XX的发展
- 近年来
- 越来越多人/越来越多XX
- 在这个时代
- 相信大家/大家都知道
- 我们都知道
- 相信大家都知道
- 在日常生活中
- 今天我想和大家分享
- 说到XX
- 提起XX
- 关于XX
- 众所周知
- 毫无疑问
- 不可否认

二、结尾客套（立即删除）
- 希望对大家有帮助
- 希望能帮到各位
- 希望能帮助到你
- 感谢阅读
- 感谢观看
- 今天的分享就到这里
- 以上就是今天的XX
- 大家有什么问题可以留言
- 欢迎评论区留言讨论
- 如果你有什么想法
- 欢迎大家补充
- 希望大家喜欢
- 谢谢大家

三、营销词（必须删除或替换）
- 强烈推荐 → 删除或换为"去看了看"
- 首选 → 删除
- 最好 → 删除或换为"还行"
- 必去 → 删除
- 强烈建议 → 删除
- 不容错过 → 删除
- 值得一试 → 删除
- 超值 → 删除
- 良心推荐 → 删除
- 宝藏店铺/医院 → 删除
- 性价比高 → 删除或换为"价格还可以"
- 效果显著 → 删除或换为"感觉有改善"
- 专业可靠 → 删除或换为"看起来挺专业"
- 服务周到 → 删除或换为"服务还行"
- 环境优美 → 删除或换为"环境还可以"

四、虚假不知所措（替换参考）
- "不知道怎么说" → 删除
- "不知道该怎么办" → "还没定好"
- "也不清楚该怎么选" → "还在对比"
- "到底要不要去" → "还在犹豫"
- "求助/急急急" → 删除
- "好纠结" → "有点犹豫"
- "不知道选哪个" → "还在比较"
- "不知道好不好" → "还在了解"
- "该不该去" → "还在考虑"

五、过于工整的句式（拆分或改写）
- "不仅...而且..."（连续出现）
- "一方面...另一方面..."
- "首先...其次...最后..."（超过3项）
- "既...又..."（连续出现）
- "可以说...也可以说..."
- "总的来说..."
- "事实上..."
- "实际上..."
- "需要注意的是..."
- "值得一提的是..."
- "从某种程度上来说..."

六、泛化词（删除或具体化）
- "大家" → "我身边有人"或删除
- "很多人" → 具体人数或删除
- "一些人" → 具体例子或删除
- "有人说" → 具体来源或删除
- "很多时候" → 具体时间或删除
- "一般来说" → 具体情况或删除
- "基本上" → 具体程度或删除
- "大多数人" → 具体群体或删除

七、AI特征词（立即删除）
- 生成
- 模型
- 算法
- 训练
- 数据
- 智能
- 人工智能
- 机器学习
- 深度学习
- 神经网络

八、替换建议
- "需要注意的是" → "我发现"
- "值得一提的是" → "另外"
- "从某种程度上来说" → "不过"
- "总的来说" → "反正"
- "事实上" → "其实"
- "实际上" → "说真的"
- "可以说" → "感觉"
- "应该说" → "我觉得"
- "不得不说" → "确实"
- "必须承认" → "说实话"
- "据说" → "看到有人说"或删除
- "一般来说" → 删除
